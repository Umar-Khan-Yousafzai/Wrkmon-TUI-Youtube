# wrkmon

Work Monitor - A developer productivity tool for monitoring system processes.

## Installation

```bash
pip install -e .
```

## Usage

```bash
wrkmon              # Launch monitor
wrkmon search "q"   # Quick search
wrkmon play <id>    # Direct play
wrkmon queue        # Show queue
wrkmon history      # Show history
```

## Requirements

- Python 3.10+
- mpv (media player)
